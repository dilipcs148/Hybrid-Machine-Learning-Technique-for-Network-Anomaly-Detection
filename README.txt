## Frameworks
All code has been written in Python 3.6.7 and has been run on a Jupyter notebook server (6.0.1). 

Frameworks and Libraries used are

* numpy: For data wrangling. (https://docs.scipy.org/doc/numpy/user/install.html)
* pandas: For data wrangling. (http://pandas.pydata.org/pandas-docs/stable/install.html)
* seaborn: For plotting.(http://seaborn.pydata.org/installing.html)
* sklearn: For machine-learning algorithms.(http://scikit-learn.org/stable/install.html)
* matplotlib: For plotting graphs.(https://matplotlib.org/1.4.3/faq/installing_faq.html)
